    No-Cloning Theorems #

    Formal statements showing that unitary cloning maps cannot exist for selected families of pure states.

    theorem QuantumComputing.Theorems.NoCloning.no_cloning_2 (n : ℕ) :
    ¬∃ (U : Square (2 ^ n * 2 ^ n)), Matrix.isUnitary U ∧ ∀ (s : Vector (2 ^ n)), U ⬝ s ⊗ ketZeros n = s ⊗ s

    No unitary n-qubit gate with an all-zero blank register can clone every state vector.

    source
    theorem QuantumComputing.Theorems.NoCloning.no_cloning_3 (n : ℕ) :
    ¬∃ (U : Square (2 * (2 * 2 ^ n))) (f : Vector 2 → Vector (2 ^ n)), Matrix.isUnitary U ∧ ∀ (s : Vector 2), s.IsNormalized → U ⬝ s ⊗ (ket0 ⊗ ketZeros n) = s ⊗ (s ⊗ f s)

    No unitary can clone an arbitrary one-qubit state into two copies while hiding extra output in an arbitrary n-qubit garbage register.

    The register dimension is written as 2 * (2 * 2 ^ n) to match the tensor shape used by the current API and avoid carrying arithmetic casts.
